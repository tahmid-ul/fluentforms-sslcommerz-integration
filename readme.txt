An add-on plugin for Fluent Forms that adds support for the SSLCOMMERZ payment gateway.

== Description ==
A simple plugin that allows you to collect payment using any Bangladeshi payment services through SSLCOMMERZ by integrating it with FluentForms Pro.

== Installation ==
You would need to have FluentForms Pro installed on your site in order to use this addon as the payment feature is only available in the Pro version of FluentForms. After you have installed FluentForms Pro, you can install this addon from the WordPress plugin repository.
